Once More With Scoobies

Hello, LaRena here! I'm a software engineer and an avid Buffy the Vampire Slayer Fan. I'm also a big fan and long time listener of the popular Buffy rewatch pod - Buffering the Vampire Slayer. Perhaps the most iconic episode of Buffy is the musical episode (S06E7) - Once More With Feeling. For this important episode, Buffering hosted a free live watch on Crowdcast on 1/30/21 that had over 1000 viewers! So some data was gathered via an optional survey and here-in lies that data all visualized and beautiful for the world to see!
Did you somehow stumble upon this project and you aren't familiar with Buffering the Vampire Slayer? You should check them out.

Slayers are awakening everywhere...

Over 500 people who participated in the livewatch completed the optional survey and we truly have slayers everywhere around the globe! We stand up, we fight evil (and the patriarchy) daily, we watch a lot of Buffy.

Zoom in, zoom out, explore, stakes are clickable! Can you find yourself?

About the Developer

Created by scooby LaRena Iocco. I'm a software engineer and have been obsessed with Buffy since 1997. I built this little project using HTML, JavaScript and CSS along with the Google Maps JavaScript API, the Charts.JS library and Bootstrap.

I have hopes of converting this to an Express.js app, but I am still struggling with some concepts in that framework, so if you happen to see this project and want to pair program with me on Express.js, I'd love to debug some things and learn from you!
